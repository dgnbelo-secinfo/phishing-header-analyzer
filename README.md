# Phishing Header Analyzer

Python tool developed to analyze suspicious email headers and identify common phishing indicators.

## Features

* IP extraction
* SPF validation check
* DKIM validation check
* DMARC validation check
* Basic spoofing detection

## Technologies

* Python 3
* Regex

## Usage

```bash
python phishing_analyzer.py
```

## Sample Detection

* SPF Fail
* DKIM Fail
* DMARC Fail
* Suspicious sender identification

## Objective

This project was created for cybersecurity portfolio purposes, focusing on SOC analysis and phishing investigation workflows.
